# Accented

A minimal static blog.

## Structure

- `index.html` — homepage
- `about.html` — about page
- `404.html` — not found page
- `css/style.css` — styles
- `posts/` — essays

## Deploy

This site is designed to be deployed as a static site with Cloudflare Pages.

Framework preset: None  
Build command: leave empty  
Output directory: `/`
